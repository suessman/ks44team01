1. 예약자 ID 클릭 : 예약자 기본정보를 보여준다.
2. 숙소상세 클릭 : 숙소 상세정보를 보여준다.
3. 결제코드 클릭 : 결제 상세내역을 보여준다. 결제취소된 경우에는 결제취소 텍스트가 나온다. 결제취소요청 클릭 후 승인을 누르면 결제취소완료가 된다.
